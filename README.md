# 🎨 Inkora

> Create, sketch, paint, and bring your ideas to life with Inkora.

Inkora is a modern digital painting application for Android designed for artists, illustrators, designers, and creative minds. Built with **Kotlin** and **Jetpack Compose**, Inkora provides a smooth, intuitive, and powerful drawing experience with a clean Material Design 3 interface.

---

## ✨ Features

- 🎨 Smooth and responsive drawing engine
- 🖌️ Multiple customizable brushes
- 📑 Layer management
- 🎨 Advanced color picker
- 📁 Gallery management
- 💾 Export artwork to PNG & JPEG
- 🌙 Beautiful Material 3 UI
- ⚡ Fast and lightweight performance
- 📱 Optimized for Android devices
